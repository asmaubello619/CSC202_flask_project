# Clinic Queue Manager

This project is a simple Flask web application that manages a clinic patient queue.

Features
- Register patients
- View waiting list
- Call next patient
- Track number of patients seen
- Timestamp patient registration

Technologies
- Python
- Flask
- Object Oriented Programming
- Queue Data Structure
- datetime module

## Project Structure


clinic_queue_manager/
├─ app.py
├─ models.py
├─ templates/
│ ├─ welcome.html
│ ├─ dashboard.html
├─ static/
│ └─ style.css

How to Run

1. Install Flask
pip install flask

2. Run the app
python app.py

3. Open browser
http://127.0.0.1:5000
